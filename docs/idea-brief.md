# 标枪单词对战 — Idea Brief

> 详细设计 prompt pack 见同目录 [`GAME_PROMPTS.md`](./GAME_PROMPTS.md)
> 本文件是 1 页电梯版，方便快速理解。

## 一句话
输入英文单词 = 释放攻击标枪，把"背单词"和"动作对战"缝起来的轻量游戏。

## 核心循环
中文释义 → 输入英文 → 标枪飞向敌人 → 命中扣血（伤害 = 词长度）→ 连击眩晕 → 胜负

## 三种模式
1. **训练模式** — 无战斗压力，连续命中 20 个词，专注肌肉记忆
2. **PVE 闯关** — 敌人有血条 + 自动攻击，60s 内击倒
3. **PVP** — 本地双人或在线 PK（即将上线）

## 核心机制
- 伤害公式：`damage = word.length × (combo >= 5 ? 1.5 : 1)`
- 连击阈值：x3 眩晕 2 秒，x5 伤害 +50%，x10 全屏特效
- 输入超时（8s）= 失败

## 美术
低多边形 + 霓虹轮廓，移动端优先。所有几何程序生成，无外部资源。

## 技术栈
- 浏览器原生 + ES modules，**零构建**
- Three.js (v0.160) 通过 importmap CDN
- localStorage 持久化（进度 / 词库）

## 里程碑

- M1：连击特效 / Boss / 音效
- M2：皮肤系统 + 商店
- M3：PVP
